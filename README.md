# AI Web Search Agent using LangChain + Gemini

This project is an autonomous AI agent that can answer questions using real-time internet search.

## Features
- Gemini LLM reasoning
- Tavily real-time web search
- Tool-using AI agent
- Autonomous decision making

## Tech Stack
- Python
- LangChain
- Google Gemini API
- Tavily Search API

## How to Run
1. Clone repo
2. Install requirements
3. Add API keys in .env
4. Run:
python app.py
## Example
User: Latest AI news  
Agent: Searches web and explains results
